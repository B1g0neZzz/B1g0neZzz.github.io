二分模板

选择模板的更新方式，如果 check(mid) 为 true，看 l = mid 还是 r = mid，如果是前者，则 mid = l + r + 1 >> 1

为何要补 mid = l + r + 1 >> 1？

如果 l = r - 1，也就是 l 与 r 只差 1时，如果 mid = l + r >> 1，那么向下取整应该 mid = l，如果 check(mid) = true, 那么又会 l = mid = l，会死循环
