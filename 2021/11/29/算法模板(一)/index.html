选择模板的更新方式，如果 check(mid) 为 true，看 l = mid 还是 r = mid，如果是前者，则 mid = l + r + 1 >> 1

为何要补 mid = l + r + 1 >> 1？

如果 l = r - 1，也就是 l 与 r 只差 1时，如果 mid = l + r >> 1，那么向下取整应该 mid = l，如果 check(mid) = true, 那么又会 l = mid = l，会死循环

二进制

n 的二进制表示第 k 位是几

① 先把第 k 位移到最后一位 n >> k

② 看个位是几 x & 1

lowbit(x)：返回 x 的最后一位 1

x = 1010，则 lowbit(x) = 10

x = 101000，则 lowbit(x) = 1000

在计算机中， -x 使用补码来表示的

x & -x = x & (~x + 1)

所以 lowbit(x) = x & (~x + 1)

典型应用： 统计 x 中 1 的个数。
