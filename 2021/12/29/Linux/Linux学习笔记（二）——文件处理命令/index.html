Linux学习笔记（二）——文件处理命令

发表于 2021-12-29 更新于 2022-01-24 分类于 Linux

Linux 常用命令

文件处理命令

命令格式

命令格式： 命令 [-选项] [参数]

​ 例： ls -la /etc

ls 命令

作用：显示目录文件

在 Linux 中，~ 表示用户的家目录

对于一般用户，~ 表示 /home/ (用户名)

对于 root 用户，~ 表示 /root

ls -a 可以看到隐藏文件

ls -l

l 代表 long

第一列 -rw 等代表权限

第二列代表引用计数

第三列代表文件所有者

第四列代表文件所属组

第五列代表文件大小

第六列代表文件最后修改时间

最后一列代表文件名

-rw-r–r–

  • 第一个字符 - 文件类型（- 二进制文件 d 目录 l 软链接文件）

  • rw- r-- r--

​ u g o

​ u 所有者，g 所属组，o 其他人

​ r 读，w 写， x 执行

ls -d

查看某个目录的信息

ls -i

目录处理命令

mkdir

这种情况，使用 -p 可以递归创建

pwd

print work directory

rmdir

remove empty directories

删除空目录

cp

copy

cp -rp [源文件或目录] [目标目录]

-r 复制目录

-p 保留文件属性

如果是文件，直接 cp 源文件 目标目录 即可

不加 -p

加了 -p

cp 命令还可以复制时修改，如下，gzhh 文件夹名称改为 cjj

mv

move

相当于剪切

可以实现改名

rm

remove

rm -rf [文件或目录]

-r 删除目录

-f 强制执行 force

touch

创建空文件

例：touch Japanlove.list

cat

查看一个文件的内容

-n 显示行号

more

如果文件比较长的话，可以使用 more 来分页显示文件内容

有几个子命令：

（空格）或 f：翻页

Enter：换行

q 或 Q：退出

b：向上翻页

less

和 more 差不多，不过用 page up 键向上翻页，上箭头可以向上翻一行。

/ 加关键词可以搜索

但是文件中可能不止这一页有关键词，可以用 n 来找下一个

head

显示文件前面几行

-n 表示行数

默认显示前 10 行

tail

看文件的后几行

tail -f 可以动态更新

链接

ln

命令英文原意：link

ln -s [原文件] [目标文件]

创建软链接

ln -s /etc/issue /tmp/issue.soft

创建文件 /etc/issue 的软链接到 /tmp/issue.soft

ln /etc/issue /tmp/issue.hard

创建文件 /etc/issue 的硬链接到 /tmp/issue.hard

软链接

软链接特征：类似 Windows 快捷方式（使用场景）

特点1：

软链接文件的所有执行权限都是 rwx

特点2：

软链接的文件大小都很小

特点3：

软链接都有一个箭头指向原文件

硬链接

特征1：

拷贝 cp -p + 同步更新

特征2：

通过 i 节点来分辨是否是硬链接

特征3：

不能跨分区

特征4：

不能针对目录使用
