5.最长回文子串

  • 动态规划

  1. 确定 dp 数组以及下标的含义

    布尔类型的 dp[i][j]: 表示区间范围 [i, j] （注意是左闭右闭）的子串是否是回文子串，如果是 dp[i][j] 为 true，否则为 false

  2. 确定递推公式

    在确定递推公式时，就要分析如下几种情况。

    整体上是两种，就是 s[i] 与 s[j] 相等，s[i] 与 s[j] 不相等这两种。

    当 s[i] 与 s[j] 不相等，那没啥好说了，dp[i][j] 一定是 false。

    当 s[i] 与 s[j] 相等时，这就复杂一些了，有如下三种情况

    • 情况一：下标 i 与 j 相同，同一个字符例如 a，当然是回文子串
    • 情况二：下标 i 与 j 相差为 1，例如 aa，也是回文子串
    • 情况三：下标 i 与 j 相差大于 1 的时候，例如 cabac，此时 s[i] 与 s[j] 已经相同了，我们看 i 到 j 区间是不是回文子串就看 aba 是不是回文就可以了，那么 aba 的区间就是 i+1 与 j-1 区间，这个区间是不是回文就看 dp[i + 1][j - 1] 是否为 true。

10.正则表达式匹配

  • 动态规划

如果不考虑 * 通配符，面对两个待匹配字符 s[i] 和 p[j]，我们唯一能做的就是看他俩是否匹配：

如果加入 * 通配符，局面就会稍微复杂一些。

当 p[j + 1] 为 * 通配符时，我们分情况讨论下：

  1. 如果匹配，即 s[i] == p[j]，那么有两种情况：
  • p[j] 有可能会匹配多个字符，比如 s = "aaa", p = "a*"，那么 p[0] 会通过 * 匹配 3 个字符 "a"。
  • p[j] 也有可能匹配 0 个字符，比如 s = "aa", p = "a*aa"，由于后面的字符可以匹配 s，所已 p[0] 只能匹配 0 次。

  1. 如果不匹配，即 s[i] != p[j]，只有一种情况：

    p[j] 只能匹配 0 次，然后看下一个字符是否能和 s[i] 匹配。比如说 s = "aa", p = "b*aa"，此时 p[0] 只能匹配 0 次。
