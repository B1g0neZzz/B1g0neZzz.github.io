vptr 和 vtbl

先看，class A，因为 A 有虚函数，所有一根虚指针 vptr 指向一个虚表 vtbl。vtbl 中放的都是函数指针。A 中有两个虚函数，所以有 vtabl 中有两个指针分别指向 A::vfunc1 和 A::vfunc2。

class B 中，推翻了 A 的 vfunc1，创建了自己的 vfunc1，继承了 A 的 vfunc2。

class C 中，推翻了 B 的 vfunc1，继承了 B 的 vfunc2，也就是 A 的 vfunc2。
