Linux学习笔记（四）——文件搜索命令

发表于 2022-01-10 更新于 2022-01-24 分类于 Linux

文件搜索命令

find

find [搜索范围] [匹配条件]

-name

find /etc -name init

在目录 /etc 中查找文件 init

find 默认为精确查找，可以：

find /etc -name *init*

星号 * 匹配任意字符，问号 ? 匹配一个任意字符

-iname

与 -name 一样，只不过不区分大小写。

-size

find / -size +204800

在根目录下查找大于 100MB 的文件。

单位是扇区，1 扇区是 512 字节，是 0.5k。

在根目录下查找大于 100MB 的文件

+n 大于 -n 小于 n 等于

-user

find /home -user dayi

在根目录下查找所有者为 dayi 的文件

-cmin

find /etc -cmin -5

在 /etc 下查找 5 分钟内被修改过属性的文件和目录

-amin 访问时间 access

-cmin 文件属性 change

-mmin 文件内容 modify

-a

find /etc -size 163840 -a -size -204800

在 /etc 下查找大于 80MB 小于 100MB 的文件。

-a 两个条件同时满足

-o 两个条件满足任意一个即可

-exec

find /etc -name inittab -exec ls -l {} \;

在 /etc 下查找 inittab 文件并显示其详细信息

-exec/ok 命令 {} \; 对搜索结果执行操作

-inum

-type

根据文件类型查找

f 文件 d 目录 l 软链接文件

locate

在文件资料库中查找文件

但是，如果新创建的文件，还没有输入到文件资料库中，就找不到，这时候可以用 updatedb

/tmp 是临时，不在 locate 所收录的文件资料库中。

which

搜索命令所在目录及别名信息

whereis

搜索命令所在目录及帮助文档路径

whereis [命令名称]

grep

在文件中搜寻字符串匹配的行并输出

grep -iv [指定字串] [文件]

grep mysql /root/install.log

-i 不区分大小写

-v 排除指定字符串

grep -v ^# /etc/inittab

man

manual 手册

获得帮助信息

man ls

查看 ls 命令的帮助信息

man services

查看配置文件 services 的帮助信息

可能有的既有配置文件又有命令，这样可以通过 1 和 5 来选择，前者是命令帮助，后者是配置文件帮助。

man 1 passwd

man 5 passwd
