f[i][j] 表示的意义：所有在第一个序列的前 i 个子母中出现，且在第二个序列的前 j 个子母中出现的子序列的最大值。

区间 DP

状态表示：f[i][j] 表示所有将第 i 堆石子到第 j 堆石子合并成一堆石子的合并方式的最小值。

状态划分：以最后一次合并的位置来划分，如下，划分为 [i, k] 和 [k + 1, j]

状态转移： f[i, k] + f[k + 1, j] + s[j] - s[i - 1]，s[i] 为前缀和
