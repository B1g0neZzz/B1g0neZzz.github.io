Socket 创建过程

Socket 创建过程：

  • 调用 socket 函数创建 Socket。
  • 调用 bind 函数分配 IP 地址和端口号。
  • 调用 listen 函数转为可接收请求状态
  • 调用 accept 函数受理连接请求

协议族（Protocol Family）

Socket 的数据传输方式（ Type ）

  • SOCK_STREAM 面向连接的 Socket

    • 传输过程中数据不会丢失。
    • 按序传输数据。
    • 传输的数据不存在数据边界（ Boundary ）（意思是 read 和 write 不需要一一对应，主要原因是面向连接的 Socket 有缓冲区 ）。

  • SOCK_DGRAM 面向消息的 Socket

    • 强调快速传输而非传输顺序。
    • 传输的数据可能丢失也可能损毁。
    • 传输的数据有数据边界。
    • 限制每次传输的数据大小。
