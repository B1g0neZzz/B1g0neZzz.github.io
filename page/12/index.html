发表于 2021-09-08 更新于 2021-12-20 分类于 算法与数据结构

饼干分配

  • 如 g = [1, 2, 3]，s = [1, 1]，结果为 1
  • 如 g = [1, 2]，s = [1, 2, 3]，结果为 2

我们可以尝试将最大的饼干给最贪心的小朋友，这样做的好处是：如果我们最大的饼干可以满足最贪心的小朋友，我们留给下一个次贪心的小朋友的饼干，也是当前看来最大的饼干。从另外一方面，如果当前最大的饼干都不能满足最贪心的小朋友，也就是说所有的饼干都不能满足这个小朋友。

命名空间的 using 声明

使用 using 声明后就无须再通过专门的后缀去获取所需的名字了。

程序中的每个名字都需要独立的 using 声明。

头文件中通常不应该包含 using 声明。因为头文件的内容会拷贝到所有引用它的文件中去，如果头文件里有某个 using 声明，那么每个使用了该头文件的文件就都会有这个声明，可能会产生名字冲突。
